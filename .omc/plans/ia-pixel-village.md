# Pixel Village IA (Information Architecture)

> 이 문서만 수정해 Pixel Village의 구역 구조와 상호작용 흐름을 변경합니다.  
> 작성일: 2026-06-18  
> 상태: 기획 진행 중

---

## 1. 개요

- **형태**: Phaser 4 기반 픽셀아트 인터랙티브 세계
- **진입**: 메인 웹사이트의 "Pixel Village" 버튼/배너 클릭
- **조작**: 데스크탑(WASD + 방향키) / 모바일(가상 조이스틱)

---

## 2. 진입 흐름

```
메인 웹사이트
    ↓  [Pixel Village 입장하기] 클릭
언어 선택 모달
    ┌─────────────────────────────┐
    │  한국어로 계속하시겠습니까?  │
    │  [한국어로 계속]  [English]  │
    └─────────────────────────────┘
    ↓
로딩 화면 (BootScene — 에셋 프리로드)
    ↓
Pixel Village 메인 월드 (MainScene)
```

---

## 3. 맵 구역 구조

### 전체 맵 레이아웃

**맵 크기**: 100×80 타일 (3,200×2,560px / 타일 1개 = 32×32px)

```
┌──────────────────────┬──────────────────────┐
│                      │                      │
│    역사/소개관        │    제품/브랜드관       │
│    (40×30 타일)      │    (40×30 타일)      │
│                      │                      │
├──────────┬───────────┼──────────────────────┤
│          │  중앙 광장  │                      │
│          │ (시작 지점) │                      │
├──────────┴───────────┼──────────────────────┤
│                      │                      │
│    조직/팀관          │    채용/IR관          │
│    (40×30 타일)      │    (40×30 타일)      │
│                      │                      │
└──────────────────────┴──────────────────────┘
```

### 시작 지점
- 플레이어는 **중앙 광장**에서 시작
- 4개 구역으로 자유롭게 이동 가능
- 각 구역 입구에 안내 간판 배치

---

## 4. 구역별 상세

### 4-1. 역사/소개관
| 항목 | 내용 |
|---|---|
| 주요 콘텐츠 | 에쓰푸드 창립 역사, 비전, 미션 |
| 주요 오브젝트 | 타임라인 간판, 창립자 NPC |
| 상호작용 | 간판 → Type B 레이어 (연혁 상세) |

### 4-2. 제품/브랜드관
| 항목 | 내용 |
|---|---|
| 주요 콘텐츠 | 소시지, 스팸, 별미당 등 제품군 |
| 주요 오브젝트 | 제품 오브젝트 (통통 튀는 애니메이션) |
| 상호작용 | 제품 터치 → Type B 레이어 (제품 상세) |

### 4-3. 조직/팀관
| 항목 | 내용 |
|---|---|
| 주요 콘텐츠 | 부서별·팀별 소개 |
| 주요 오브젝트 | 직원 NPC (팀별 구분 캐릭터) |
| 상호작용 | NPC 대화 → Type A 텍스트박스 |

### 4-4. 채용/IR관
| 항목 | 내용 |
|---|---|
| 주요 콘텐츠 | 채용 공고, 투자자 정보 |
| 주요 오브젝트 | 채용 안내판, IR 서류 오브젝트 |
| 상호작용 | 안내판 → Type B 레이어 (공고 상세) |

---

## 5. 상호작용 레이어 타입

| 타입 | 형태 | 용도 |
|---|---|---|
| **Type A** | 화면 하단 RPG 텍스트박스 | 짧은 NPC 대화, 힌트 |
| **Type B** | 화면 50% 슬라이드 레이어 | 상세 설명 + 이미지 |
| **Type C** | 풀스크린 레이어 | 영상 (브랜드필름, 광고) |

### 상호작용 트리거
- **데스크탑**: 인터랙션 존 진입 후 `Enter` 키
- **모바일**: 인터랙션 존 진입 후 화면 탭

---

## 6. NPC & 오브젝트 배치

| 유형 | 배치 위치 | 이동 방식 | 상호작용 |
|---|---|---|---|
| 직원 NPC | 조직/팀관 중심 | 랜덤 배회 | ✅ Type A 대화 |
| 동물 (소/돼지/오리) | 전 구역 랜덤 | 랜덤 배회 | ✅ 반응 (이스터에그) |
| 제품 오브젝트 | 제품/브랜드관 | 고정 (애니메이션) | ✅ Type B |
| 간판 | 구역 입구, 역사관 | 고정 | ✅ Type A 또는 B |
| 장식 요소 | 전 구역 | 배경 연출 | ❌ |

---

## 7. 나가기 / 메인 복귀

- 인게임 ESC 메뉴 또는 상단 고정 버튼으로 메인 웹사이트 복귀
- 브라우저 뒤로가기도 동일하게 처리
