# Pixel Village IA (Information Architecture)

> 이 문서만 수정해 Pixel Village의 구역 구조와 상호작용 흐름을 변경합니다.  
> 작성일: 2026-06-18  
> 상태: 기획 진행 중

---

## 1. 개요

- **형태**: Phaser 4 기반 픽셀아트 인터랙티브 세계
- **진입**: 메인 웹사이트의 "Pixel Village" 버튼/배너 클릭
- **조작**: 데스크탑(WASD + 방향키) / 모바일(가상 조이스틱)

---

## 2. 진입 흐름

```
메인 웹사이트
    ↓  [Pixel Village 입장하기] 클릭
언어 선택 모달
    ┌─────────────────────────────┐
    │  한국어로 계속하시겠습니까?  │
    │  [한국어로 계속]  [English]  │
    └─────────────────────────────┘
    ↓
로딩 화면 (BootScene — 에셋 프리로드)
    ↓
Pixel Village 메인 월드 (MainScene)
```

---

## 3. 맵 구역 구조

> 레퍼런스: 에쓰푸드 브랜드 월드맵 일러스트 (IMG_5510)  
> 전체 분위기: 녹색 푸른 목장, 아이소메트릭 Top-down RPG

### 전체 맵 레이아웃

**맵 크기**: 100×80 타일 이상 (레퍼런스 대비 확장 필요)

```
┌─────────────────────────────────────────────────────────────────┐
│  [하늘] 푸른 하늘 + 뭉게구름 + ✈️ 비행기(수입상품 영역)           │
│  [산] 좌측: "BETTER FOOD BETTER WORLD" 헐리우드 간판            │
│       우측: 산 + 지중해(바다)                                    │
├──────────────────┬──────────────────┬───────────────────────────┤
│  ③ 로젠치즈앤푸드  │   ⑪ 마스터즈브렌드  │  ⑨ 메디쏠라               │
│  (공장+치즈문)    │   (빵 공장)        │  (지중해·산 걸침)          │
├──────────────────┼──────────────────┼───────────────────────────┤
│  ① 소 목장 영역   │  ② 팜프레시웨이브  │  ⑧ 존쿡&사퀴테리           │
│  (설성목장·SFarm) │  (소↔돼지 사이)   │  (오프라인 지점)           │
├──────────────────┼──────────────────┼───────────────────────────┤
│  ④ 돼지 방목구역  │  ⑤ 연못 영역      │  ⑦ 음성공장 영역           │
│  (미트멜로우·     │  (설성 오리)      │  (메인 공장)               │
│   몬테사노)       ├──────────────────┤                           │
│                  │  ⑥ 존쿡델리미트   │                           │
│                  │  (B2C 브랜드)     │                           │
├──────────────────┼──────────────────┼───────────────────────────┤
│  ⑩ 프레스그린     │  ⑮ 스마트푸드네트웍스 트럭 (길 연결)            │
│  (착즙·주스)      ├──────────────────┬───────────────────────────┤
│                  │  ⑬ 도담도담       │  ⑭ OWP                    │
│                  │  (급식 브랜드)    │  (미국 계열사)             │
└──────────────────┴──────────────────┴───────────────────────────┘
```

### 시작 지점
- 플레이어는 **팜프레시웨이브 또는 중앙 광장**에서 시작
- 15개 구역을 길로 이어서 자유롭게 이동
- 스마트푸드네트웍스 트럭이 길목마다 배치되어 길 안내 역할

## 4. 구역별 상세

### 4-1. 소 목장 영역
| 항목 | 내용 |
|---|---|
| 브랜드 | 설성목장, S Farm 및 하위 브랜드 |
| 분위기 | 초록 목초지, 소 NPC 배회 |
| 주요 오브젝트 | 목장 건물, 울타리, 소 NPC |
| 상호작용 | 간판/건물 → Type B (브랜드 소개) |

### 4-2. 팜프레시웨이브
| 항목 | 내용 |
|---|---|
| 위치 | 소 목장과 돼지 방목구역 사이 |
| 브랜드 | Farm Fresh Wave |
| 주요 오브젝트 | Farm Fresh 표지판, 신선식품 오브젝트 |
| 상호작용 | 표지판 → Type B (브랜드 소개) |

### 4-3. 로젠치즈앤푸드 영역
| 항목 | 내용 |
|---|---|
| 브랜드 | 로젠치즈앤푸드 (공장), 치즈문 |
| 주요 오브젝트 | 공장 건물, 치즈 오브젝트 |
| 상호작용 | 공장 → Type B (브랜드/제품 소개) |

### 4-4. 돼지 방목구역
| 항목 | 내용 |
|---|---|
| 브랜드 | 미트멜로우, 몬테사노 |
| 분위기 | 넓은 방목장, 돼지 NPC 배회 |
| 주요 오브젝트 | 방목장 울타리, 돼지 NPC |
| 상호작용 | 간판 → Type B (브랜드 소개) |

### 4-5. 연못 영역
| 항목 | 내용 |
|---|---|
| 브랜드 | 설성 오리 |
| 분위기 | 연못, 오리 NPC 유영 |
| 주요 오브젝트 | 연못, 오리 NPC |
| 상호작용 | 오리/간판 → Type A 대화 or Type B |

### 4-6. 존쿡 델리 미트 영역
| 항목 | 내용 |
|---|---|
| 브랜드 | 존쿡 델리 미트 (B2C) |
| 주요 오브젝트 | 델리 상점 건물, 햄·소시지 오브젝트 |
| 상호작용 | 상점 → Type B (B2C 브랜드 소개) |

### 4-7. 음성공장 영역
| 항목 | 내용 |
|---|---|
| 브랜드 | 에쓰푸드 메인 공장 |
| 주요 오브젝트 | 대형 공장 건물, S FOOD 로고 |
| 상호작용 | 공장 → Type B or C (공장 소개·영상) |

### 4-8. 존쿡&사퀴테리 영역
| 항목 | 내용 |
|---|---|
| 브랜드 | 존쿡, Charcutería (사퀴테리) |
| 주요 오브젝트 | 유럽풍 상점 건물 |
| 상호작용 | 상점 → Type B (오프라인 지점 소개) |

### 4-9. 메디쏠라 영역
| 항목 | 내용 |

| 브랜드 | 메디쏠라 |

| 분위기 | 지중해 느낌, 산기슭 |

### 4-10. 프레스그린 영역

### 4-11. 마스터즈브렌드 영역

| 상호작용 | 공장 → Type B (브랜드·제품 소개) |

### 4-12. 하늘 — 비행기 영역

| 상호작용 | 비행기 터치 → Type B (수입상품 소개) |

| 상호작용 | 트럭 → Type A 대화 (회사 소개 한 줄) |

## 5. 상호작용 레이어 타입

| 타입 | 형태 | 용도 |
|---|---|---|
| **Type A** | 화면 하단 RPG 텍스트박스 | 짧은 NPC 대화, 힌트 |
| **Type B** | 화면 50% 슬라이드 레이어 | 상세 설명 + 이미지 |
| **Type C** | 풀스크린 레이어 | 영상 (브랜드필름, 광고) |

### 상호작용 트리거
- **데스크탑**: 인터랙션 존 진입 후 `Enter` 키
- **모바일**: 인터랙션 존 진입 후 화면 탭

---

## 6. NPC & 오브젝트 배치

| 유형 | 배치 위치 | 이동 방식 | 상호작용 |
|---|---|---|---|
| 소 NPC | 소 목장 영역 | 랜덤 배회 | ✅ 이스터에그 |
| 돼지 NPC | 돼지 방목구역 | 랜덤 배회 | ✅ 이스터에그 |
| 오리 NPC | 연못 영역 | 유영 | ✅ Type A 대화 |
| 스마트푸드 트럭 NPC | 구역 사이 길목 | 고정 or 순찰 | ✅ Type A 한 줄 소개 |
| 비행기 오브젝트 | 하늘 레이어 | 좌→우 이동 (루프) | ✅ Type B 수입상품 소개 |
| 브랜드 건물/공장 | 각 구역 | 고정 | ✅ Type B 브랜드 소개 |
| 간판 | 구역 입구 | 고정 | ✅ Type A 또는 B |
| 장식 요소 (구름, 꽃 등) | 전 구역 | 배경 연출 | ❌ |
