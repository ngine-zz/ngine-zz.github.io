# 에쓰푸드 웹사이트 프로젝트 전체 개요

> 작성일: 2026-06-18  
> 상태: 기획 진행 중

---

## 1. 전체 구조

에쓰푸드 웹사이트는 **2개의 독립된 경험**으로 구성됩니다.

```
https://sfood.com (도메인 접속)
        ↓
┌─────────────────────────────────────────────────┐
│           메인 웹사이트 (일반 반응형)              │
│  - 회사 소개, 제품, 채용, IR 등 표준 웹 구조      │
│  - 모바일/데스크탑 완전 반응형                    │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │  [ Pixel Village 입장하기 ]  ← 버튼/배너 │   │
│  └──────────────────┬──────────────────────┘   │
└─────────────────────┼───────────────────────────┘
                      ↓ 클릭
┌─────────────────────────────────────────────────┐
│           Pixel Village (픽셀아트 인터랙티브)      │
│  - Phaser 4 기반 픽셀아트 세계                   │
│  - 캐릭터 이동, NPC 상호작용, 구역 탐험           │
│  - 데스크탑(키보드) + 모바일(가상 조이스틱)        │
└─────────────────────────────────────────────────┘
```

---

## 2. 컴포넌트별 역할 분리

| 항목 | 메인 웹사이트 | Pixel Village |
|---|---|---|
| 목적 | 정보 전달 (SEO 중심) | 경험/몰입 (인터랙티브) |
| 기술 | Angular (일반 반응형) | Angular + Phaser 4 |
| 조작 | 마우스/터치 스크롤 | WASD + 조이스틱 |
| 콘텐츠 | 정형화된 텍스트/이미지 | NPC 대화, 탐험, 발견 |
| SEO | ✅ 필수 | ❌ 불필요 |
| 모바일 | 완전 반응형 | 가상 조이스틱 지원 |

---

## 3. Pixel Village 진입점

메인 웹사이트에서 Village로 연결하는 UI 요소 (확정 필요):

- 메인 홈 화면 배너
- 상단 GNB 메뉴 항목
- 특정 섹션 내 버튼

> 진입 방식 및 위치는 메인 웹사이트 IA 확정 후 결정합니다.
